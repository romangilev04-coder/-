---------------------------------------------------------------------------------------------------------------------------------------
Добро пожаловать в персональный мини-ежедневник, созданный для удобного хранения и управления вашими задачами!
Это простое, лёгкое и полностью локальное приложение, которое работает прямо в браузере — без серверов, без аккаунтов и без интернета.
Ежедневник поможет вам не забывать важные дела, структурировать день и хранить задачи в удобном и понятном формате.
Возможностями данного ежедневника являются: добавление задач, автосохранение, удаление задач, адаптивный интерфейс и мгновенная работа!
----------------------------------------------------------------------------------------------------------------------------------------
Как пользоваться данным ежедневником?
Откройте файл index.html в любом браузере (Chrome, Edge, Firefox и т.д.).
Введите задачу и нажмите кнопку добавить — она появится ниже в виде карточки. При необходимости укажите описание.
Чтобы удалить задачу, нажмите «Удалить» на карточке.
----------------------------------------------------------------------------------------------------------------------------------------
Все задачи сохраняются автоматически, так что при следующем запуске ежедневника вы увидите всю историю своих дел.
